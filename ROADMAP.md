# PosterFlow AI Roadmap

PosterFlow AI 的长期定位是：面向大学生设计师和个人接单者的 AI 接单与交付工作流助手。

项目采用小步迭代：每次只完成一个可验证能力，测试通过后再提交 GitHub。

## v1.0.0 — 稳定业务底座

- [x] 客户、订单、报价、修改和付款管理
- [x] 图片元数据与实际印刷 DPI 检测
- [x] 经营数据看板
- [x] SQLite 本地持久化
- [x] 自动化测试、演示数据和 GitHub Actions
- [x] 隐私保护与 GitHub 发布整理

## v1.1.0 — AI 需求助手

- [ ] 将客户自然语言需求整理为结构化字段
- [ ] 标记缺失、冲突或不确定的信息
- [ ] 自动生成需要向客户追问的问题
- [ ] 结合规则报价生成可解释的价格说明
- [ ] 生成客户需求确认与报价话术
- [ ] 建立匿名需求样本和字段提取评测

## v1.2.0 — 修改与版本交付

- [ ] 上传和管理 V1、V2、终稿等设计版本
- [ ] 将客户修改意见拆解为可勾选任务
- [ ] 自动识别超出免费次数的修改
- [ ] 对比版本信息并保留交付记录
- [ ] 生成最终交付检查清单

## v1.3.0 — 文档与经营分析

- [ ] 生成报价单和交付报告
- [ ] 分析客户来源、客单价、回款和修改成本
- [ ] 识别高修改、高逾期和低利润订单
- [ ] 导出匿名经营数据

## v2.0.0 — 在线产品化

- [ ] 登录注册与多用户数据隔离
- [ ] PostgreSQL、对象存储和数据库迁移
- [ ] 在线部署、监控和备份
- [ ] 根据真实用户反馈评估 Vue 3 + FastAPI 重构

## 每日迭代规则

1. 从路线图选择一个最小任务。
2. 完成功能或文档改动。
3. 运行 `python -m pytest -q`。
4. 更新 CHANGELOG 或路线图状态。
5. 使用清晰的提交信息推送 GitHub。

每日完成一个小闭环，比一次堆叠大量未验证功能更重要。

# Changelog

PosterFlow AI 的重要变更记录在此文件中。版本号遵循语义化版本规则：
`主版本.功能版本.修订版本`。

## [1.0.0] - 2026-07-19

### Added

- 客户建档、来源记录与历史订单查看
- 订单创建、搜索、筛选、状态流转与删除
- 基于设计类型、加急、印刷和复杂度的规则报价
- 免费修改次数、额外修改费和修改状态管理
- PNG、JPG、WEBP 元数据与实际印刷 DPI 检测
- 设计版本、付款、退款、应收与实收记录
- 订单、客户来源、设计类型和月度收入数据看板
- SQLite 本地持久化和 10 条幂等演示数据
- 数据库、报价、图片检测与密码安全自动化测试
- GitHub Actions 持续集成工作流

### Security

- 本地数据库、上传文件、环境变量和密钥默认不进入 Git
- 上传文件限制格式与体积
- 密码工具采用带随机盐的 PBKDF2-SHA256

### Notes

- v1.0.0 是稳定业务流程底座，尚未接入大模型。
- AI 需求解析、缺失信息追问和客户确认话术列入 v1.1.0。
